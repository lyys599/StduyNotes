[[第9章_欧几里得空间]]

## 模块 0：概念的宏观定位

在欧氏空间中，**正交投影**不仅仅是一个几何动作，它是连接“全空间 $V$”与“局部子空间 $W$”的桥梁。它的本质是：**在极其复杂的全空间中，滤掉所有与 $W$ 无关（正交）的噪音，提取出向量在 $W$ 这个局部参考系下的“最纯粹的信息”**。它在后期的最小二乘法、傅里叶级数（函数空间的投影）中起着决定性作用。

## 模块 1：核心概念破拆

### 1. 正交投影的代数定义

依据核心定理，全空间可以进行正交直和分解 $V = W \oplus W^\perp$。

对于空间中任意给定的向量 $\alpha \in V$，它都可以**唯一地**拆解为两部分之和：

$$\alpha = \alpha_W + \alpha_{W^\perp}$$

其中 $\alpha_W \in W$，且 $\alpha_{W^\perp} \in W^\perp$。

这里的 **$\alpha_W$** 就被称为向量 $\alpha$ 在子空间 $W$ 上的**正交投影**；而 **$\alpha_{W^\perp}$** 称为 $\alpha$ 到 $W$ 的**垂线段**（或正交分量）。

### 2. 高维几何透镜（直觉比喻）

- 💡 **几何画面**：想象子空间 $W$ 是一张水平的“桌面”，向量 $\alpha$ 是一根斜插在桌面上的“木棍”。此时，一束极其强烈的阳光从正上方（沿着 $W^\perp$ 的方向）垂直照射下来。
    
- 💡 **对应关系**：
    
    - 桌面上木棍的**绝对平直的影子**，就是正交投影 $\alpha_W$。
        
    - 光线垂直照射的**光柱轨迹**（从木棍顶端垂直砸向桌面的线），就是正交分量 $\alpha_{W^\perp}$。
        
    - 勾股定理在此时完美成立：$|\alpha|^2 = |\alpha_W|^2 + |\alpha_{W^\perp}|^2$。
        

## 模块 2：做题 SOP（正交投影的计算程序）

在考场上，已知向量 $\alpha$ 和子空间 $W$（通常由一组基生成），如何手算出这个影子 $\alpha_W$？请严格遵守以下 3 步绝对安全的计算流：

**SOP 执行流程：**

1. **寻找正交基（极其关键）**：不要急着代公式！先检查题目给你的 $W$ 的基底 $\alpha_1, \alpha_2, \dots, \alpha_k$ 是否两两正交。如果不是，**必须先使用 Schmidt 正交化**将它们化为一组标准正交基 $\epsilon_1, \epsilon_2, \dots, \epsilon_k$。
    
2. **内积拼装（投影公式）**：利用标准正交基，套用投影的线性组合公式直接算出影子：
    
    $$\alpha_W = (\alpha, \epsilon_1)\epsilon_1 + (\alpha, \epsilon_2)\epsilon_2 + \dots + (\alpha, \epsilon_k)\epsilon_k$$
    
    _(这里的每一个系数 $(\alpha, \epsilon_i)$，物理学上叫做 $\alpha$ 在各个坐标轴上的“傅里叶系数”或“投影分量”。)_
    
3. **验算防伪**：求出 $\alpha_W$ 后，如果你要求垂线段，直接用减法 $\alpha_{W^\perp} = \alpha - \alpha_W$。为了确保算对，抽查内积 $(\alpha_{W^\perp}, \alpha_W)$，如果等于 0，说明彻底算对！
    

## 模块 3：骨干定理与思想升华

> [!abstract] 核心定理：最佳逼近定理 (最小距离定理)
> 
> 设 $\alpha_W$ 是 $\alpha$ 在子空间 $W$ 上的正交投影，那么对于 $W$ 中的**任意**其他向量 $\beta$，恒有：
> 
> $$|\alpha - \alpha_W| \le |\alpha - \beta|$$
> 
> 等号成立当且仅当 $\beta = \alpha_W$。
> 
> **思想升华**：正交投影是全空间中的点到子空间 $W$ **距离最短的点**。这就是为什么在数据分析中，“最小二乘法”的本质就是求一个多维向量在样本空间上的正交投影——因为它能让误差（垂线段的长度）降到绝对的最低！

## 模块 4：易错陷阱库

> [!warning]- 易错点 1：用斜交基底强行套用投影公式
> 
> - **错误直觉**：已知 $W = L(\beta_1, \beta_2)$，求 $\alpha$ 的投影时，直接写出 $\alpha_W = c_1\beta_1 + c_2\beta_2$，然后把 $c_i$ 当作 $(\alpha, \beta_i)$ 乱算。
>     
> - **正确纠偏**：前面的 SOP 公式 $\sum (\alpha, \epsilon_i)\epsilon_i$ **极其依赖基底的正交性**！如果基底不是正交的，你算出来的“影子”是歪的，它不属于正交投影！必须先用 Schmidt 方法把 $\beta_1, \beta_2$ 变直！
>     

> [!warning]- 易错点 2：混淆“投影”与“投影矩阵”的迹象
> 
> - **错误直觉**：认为投影的长度 $|\alpha_W|$ 会大于原向量的长度 $|\alpha|$。
>     
> - **正确纠偏**：由于勾股定理 $|\alpha|^2 = |\alpha_W|^2 + |\alpha_{W^\perp}|^2$，且长度的平方非负，因此绝对有 $|\alpha_W| \le |\alpha|$。投影是一个“信息压缩（降维）”的过程，影子的长度永远不可能超过木棍本身的长度（除非木棍原本就平躺在桌面上，此时取等号）。


### 1. 几何直觉：垂直落下的“影子”

想象在三维空间中，有一个二维的桌面（把它看作子空间 W），以及桌面外悬浮着的一个箭头（向量 α）。 现在，你在桌面的正上方垂直向下打一束光，这个悬浮的箭头在桌面上投下了一个影子。 这个**影子**，就是向量 α 在子空间 W 上的**正交投影**。

### 2. 严格的代数定义：空间的“正交切割”

在代数上，我们不谈论光和影子，我们谈论“空间的分解”。

设 W 是欧氏空间 V 的一个子空间。根据上一章的知识，空间 V 可以被完美地切分为 W 和它的正交补 W⊥ 的直和：V=W⊕W⊥。

这就意味着，对于空间中的**任意**一个向量 α，它都可以被**唯一地**拆解为两个部分之和：

α=α1​+α2​

其中：

- α1​∈W （躺在桌面上的部分）
    
- α2​∈W⊥ （完全垂直于桌面的部分）
    

在这里，**α1​ 就被称为 α 在子空间 W 上的正交投影**。而 α2​ 通常被称为垂线段向量（或正交分量）。

### 3. 核心计算法则（SOP）：到底怎么算？

在做题时，我们怎么把这个“影子” α1​ 给揪出来呢？

**前提条件（最容易踩坑的地方）**：你必须先在子空间 W 中找到一组**正交基** ϵ1​,ϵ2​,…,ϵk​。（如果题目给的基不正交，你必须先用施密特正交化把它们“掰直”）。

**投影公式**： 一旦有了正交基，计算极其简单，就是求 α 在每个基向量上的“分量”然后加起来：

ProjW​(α)=α1​=(ϵ1​,ϵ1​)(α,ϵ1​)​ϵ1​+(ϵ2​,ϵ2​)(α,ϵ2​)​ϵ2​+⋯+(ϵk​,ϵk​)(α,ϵk​)​ϵk​

_(注：如果 ϵi​ 已经是标准正交基，即长度全为1，那么分母全为1，公式会变得更简单：∑(α,ϵi​)ϵi​)_

### 4. 为什么正交投影这么重要？（最短距离定理）

数学家为什么要费尽心机找这个投影？因为它解决了一个极其现实的问题：**最优近似（最小二乘法）**。

假设你站在空间中的点 α 处，你想走到桌面 W 上。你怎么走最近？ 直觉告诉你：“走垂线段最近”。

高等代数用**最短距离定理**严格证明了这一点： 在子空间 W 的所有向量中，**正交投影 α1​ 是距离 α 最近的那个向量**。 即对于 W 中的任意其他向量 β，都有：

∣α−α1​∣≤∣α−β∣

等号成立当且仅当 β=α1​。

**现实应用：** 在现实中，当一个线性方程组 Ax=b 无解时（比如测量数据有误差，导致找不到完美的解），我们不会直接放弃。我们会把向量 b **正交投影**到矩阵 A 的列空间上得到 b′，然后去解 Ax=b′。这样求出来的解虽然不是绝对完美的，但它是**误差最小的“最优解”**。这就是统治了统计学、机器学习底层逻辑的“最小二乘法”。

### 🚀 正交投影实战案例

**题目**：在 $\mathbb{R}^3$ 欧氏空间中，子空间 $W$ 由向量 $\beta_1 = (1, 1, 0)^T$ 和 $\beta_2 = (1, 0, 1)^T$ 张成。求向量 $\alpha = (1, 2, 3)^T$ 在 $W$ 上的正交投影 $\alpha_W$。

### 第 1 步：施密特正交化 (把歪基底掰直)

首先，$\beta_1$ 和 $\beta_2$ 显然不正交（内积为 1）。我们需要构造标准正交基 $\{\epsilon_1, \epsilon_2\}$。

1. **取第一个正交向量**：
    
    $\eta_1 = \beta_1 = (1, 1, 0)^T$
    
    单位化：$\epsilon_1 = \frac{\eta_1}{|\eta_1|} = \frac{1}{\sqrt{2}}(1, 1, 0)^T$
    
2. **构造第二个正交向量**：
    
    $\eta_2 = \beta_2 - (\beta_2, \epsilon_1)\epsilon_1$
    
    计算内积：$(\beta_2, \epsilon_1) = \frac{1}{\sqrt{2}}(1\cdot 1 + 0\cdot 1 + 1\cdot 0) = \frac{1}{\sqrt{2}}$
    
    $\eta_2 = (1, 0, 1)^T - \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}(1, 1, 0)^T = (1, 0, 1)^T - (\frac{1}{2}, \frac{1}{2}, 0)^T = (\frac{1}{2}, -\frac{1}{2}, 1)^T$
    
    单位化：$|\eta_2| = \sqrt{\frac{1}{4} + \frac{1}{4} + 1} = \sqrt{\frac{3}{2}}$
    
    $\epsilon_2 = \sqrt{\frac{2}{3}}(\frac{1}{2}, -\frac{1}{2}, 1)^T = (\frac{1}{\sqrt{6}}, -\frac{1}{\sqrt{6}}, \frac{2}{\sqrt{6}})^T$
    

### 第 2 步：套用投影公式 (算出影子)

现在有了标准正交基 $\{\epsilon_1, \epsilon_2\}$，计算投影 $\alpha_W = (\alpha, \epsilon_1)\epsilon_1 + (\alpha, \epsilon_2)\epsilon_2$。

1. 计算系数：
    
    $(\alpha, \epsilon_1) = (1, 2, 3) \cdot \frac{1}{\sqrt{2}}(1, 1, 0) = \frac{3}{\sqrt{2}}$
    
    $(\alpha, \epsilon_2) = (1, 2, 3) \cdot (\frac{1}{\sqrt{6}}, -\frac{1}{\sqrt{6}}, \frac{2}{\sqrt{6}}) = \frac{1-2+6}{\sqrt{6}} = \frac{5}{\sqrt{6}}$
    
2. 带入公式：
    
    $\alpha_W = \frac{3}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}(1, 1, 0) + \frac{5}{\sqrt{6}} \cdot (\frac{1}{\sqrt{6}}, -\frac{1}{\sqrt{6}}, \frac{2}{\sqrt{6}})$
    
    $\alpha_W = \frac{3}{2}(1, 1, 0) + \frac{5}{6}(1, -1, 2)$
    
    $\alpha_W = (\frac{3}{2}+\frac{5}{6}, \frac{3}{2}-\frac{5}{6}, \frac{10}{6}) = (\frac{14}{6}, \frac{4}{6}, \frac{10}{6}) = (\frac{7}{3}, \frac{2}{3}, \frac{5}{3})^T$
    

### 第 3 步：验证与结果

- **正交分量**：$\alpha_{W^\perp} = \alpha - \alpha_W = (1, 2, 3)^T - (\frac{7}{3}, \frac{2}{3}, \frac{5}{3})^T = (-\frac{4}{3}, \frac{4}{3}, \frac{4}{3})^T$
    
- **神级验算**：检查 $\alpha_{W^\perp}$ 是否与 $W$ 正交（即与 $\beta_1, \beta_2$ 内积是否为 0）：
    
    $(-\frac{4}{3}, \frac{4}{3}, \frac{4}{3}) \cdot (1, 1, 0) = -\frac{4}{3} + \frac{4}{3} = 0$ （通过！）
    
    $(-\frac{4}{3}, \frac{4}{3}, \frac{4}{3}) \cdot (1, 0, 1) = -\frac{4}{3} + \frac{4}{3} = 0$ （通过！）
    

只要基底变成了 $\epsilon_1, \epsilon_2$，投影计算就简化成了纯粹的内积乘法加和，这本质上就是你把原向量 $\alpha$ 投影到了一个个互相垂直的坐标轴上。
