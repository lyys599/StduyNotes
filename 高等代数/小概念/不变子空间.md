# 不变子空间：定义、性质与核心应用

> **核心定位**：不变子空间是线性变换 $\mathscr{A}$ 作用下"自我封闭"的子空间。它是将复杂线性变换"分而治之"的核心工具——通过将全空间分解为若干不变子空间的直和，$\mathscr{A}$ 的矩阵可化为分块对角形，每个块对应一个更简单的"子变换"。这是从一般线性变换走向对角化、Jordan标准型的关键桥梁。

---

## 一、严格定义

### 1.1 定义

设 $V$ 是数域 $P$ 上的线性空间，$\mathscr{A} \in L(V)$，$W$ 是 $V$ 的一个子空间。若

$$\mathscr{A}W \subseteq W$$

即对任意 $\alpha \in W$，都有 $\mathscr{A}\alpha \in W$，则称 $W$ 是 $\mathscr{A}$ 的**不变子空间**（invariant subspace），简称 **$\mathscr{A}$-子空间**。

> 等价表述：$W$ 是 $\mathscr{A}$-子空间 $\Leftrightarrow$ $\mathscr{A}$ 在 $W$ 上的限制 $\mathscr{A}|_W : W \to W$ 仍是线性变换。

### 1.2 矩阵语言

设 $\mathscr{A}$ 在基 $\{\varepsilon_1, \ldots, \varepsilon_n\}$ 下的矩阵为 $A$，$W$ 由前 $k$ 个基向量张成。则

$$W \text{ 是 } \mathscr{A}\text{-子空间} \Leftrightarrow A = \begin{pmatrix} A_1 & B \\ 0 & A_2 \end{pmatrix}$$

其中 $A_1$ 是 $k \times k$ 矩阵（即 $\mathscr{A}|_W$ 在 $W$ 的基下的矩阵）。

若 $V = W_1 \oplus W_2$ 且 $W_1, W_2$ 都是 $\mathscr{A}$-子空间，则

$$A = \begin{pmatrix} A_1 & 0 \\ 0 & A_2 \end{pmatrix}$$

（分块对角形）。

---

## 二、六大典型例子

### 例1：平凡不变子空间

对任意 $\mathscr{A} \in L(V)$，以下两个子空间总是 $\mathscr{A}$-子空间：

| 子空间 | 验证 | 类型 |
|:---:|:---|:---:|
| $\{0\}$ | $\mathscr{A}(0) = 0 \in \{0\}$ | 平凡 |
| $V$ | $\mathscr{A}\alpha \in V$（定义） | 平凡 |

---

### 例2：核（零空间）$\mathscr{A}^{-1}(0)$

$$W = \mathscr{A}^{-1}(0) = \{\alpha \in V \mid \mathscr{A}\alpha = 0\}$$

**验证**：设 $\alpha \in W$，则 $\mathscr{A}\alpha = 0 \in W$。$\checkmark$

> **意义**：核是"被 $\mathscr{A}$ 消灭"的最大子空间。$\mathscr{A}$ 在核上的限制是零变换。

---

### 例3：像（值域）$\mathscr{A}V$

$$W = \mathscr{A}V = \{\mathscr{A}\alpha \mid \alpha \in V\}$$

**验证**：设 $\beta \in \mathscr{A}V$，则 $\beta = \mathscr{A}\alpha$ 对某 $\alpha \in V$。

$$\mathscr{A}\beta = \mathscr{A}(\mathscr{A}\alpha) = \mathscr{A}^2\alpha \in \mathscr{A}V \;\checkmark$$

> **意义**：像中每个向量仍是像（被 $\mathscr{A}$ 作用后仍在像中）。

---

### 例4：特征子空间 $V_\lambda$

$$W = V_\lambda = \{\alpha \in V \mid \mathscr{A}\alpha = \lambda\alpha\}$$

**验证**：设 $\alpha \in V_\lambda$，则 $\mathscr{A}\alpha = \lambda\alpha$。

$$\mathscr{A}(\mathscr{A}\alpha) = \mathscr{A}(\lambda\alpha) = \lambda(\mathscr{A}\alpha) = \lambda(\lambda\alpha) = \lambda^2\alpha$$

这说明 $\mathscr{A}\alpha = \lambda\alpha \in V_\lambda$（因 $\mathscr{A}(\lambda\alpha) = \lambda(\mathscr{A}\alpha) = \lambda^2\alpha = \lambda(\lambda\alpha)$）。$\checkmark$

> **意义**：特征子空间是 $\mathscr{A}$ 在其上退化为**数乘变换**的最重要不变子空间。

---

### 例5：根子空间（广义特征子空间）

$$W = G_\lambda = \{\alpha \in V \mid (\mathscr{A} - \lambda\mathscr{E})^k\alpha = 0 \text{（某 } k \geq 1\text{）}\}$$

**验证**：设 $\alpha \in G_\lambda$，$(\mathscr{A} - \lambda\mathscr{E})^k\alpha = 0$。则

$$(\mathscr{A} - \lambda\mathscr{E})^k(\mathscr{A}\alpha) = (\mathscr{A} - \lambda\mathscr{E})^k((\mathscr{A} - \lambda\mathscr{E})\alpha + \lambda\alpha)$$
$$= (\mathscr{A} - \lambda\mathscr{E})^{k+1}\alpha + \lambda(\mathscr{A} - \lambda\mathscr{E})^k\alpha = 0 + 0 = 0$$

故 $\mathscr{A}\alpha \in G_\lambda$。$\checkmark$

> **意义**：根子空间是**最大的**含于 $V_\lambda$ 相关的不变子空间，是构造 Jordan 标准型的舞台。

---

### 例6：由特征向量生成的子空间

设 $\xi_1, \ldots, \xi_k$ 是 $\mathscr{A}$ 的特征向量（不必属于同一特征值），$W = \text{span}\{\xi_1, \ldots, \xi_k\}$。

**验证**：设 $\alpha = \sum c_i\xi_i \in W$。

$$\mathscr{A}\alpha = \sum c_i\mathscr{A}\xi_i = \sum c_i\lambda_i\xi_i \in W \;\checkmark$$

> **意义**：多个特征向量张成的子空间是不变子空间。当它们构成全空间的基时，$\mathscr{A}$ 可对角化。

## 三、八大核心性质

|   编号   |    性质    | 内容                                                                                                            | 证明要点                                                                                                         |
| :----: | :------: | :------------------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------- |
| **P1** |  交的封闭性   | $W_1, W_2$ 是 $\mathscr{A}$-子空间 $\Rightarrow$ $W_1 \cap W_2$ 是 $\mathscr{A}$-子空间                               | $\alpha \in W_1 \cap W_2$ $\Rightarrow$ $\mathscr{A}\alpha \in W_1$ 且 $\in W_2$                              |
| **P2** |  和的封闭性   | $W_1, W_2$ 是 $\mathscr{A}$-子空间 $\Rightarrow$ $W_1 + W_2$ 是 $\mathscr{A}$-子空间                                  | $\alpha = \alpha_1 + \alpha_2$，$\mathscr{A}\alpha = \mathscr{A}\alpha_1 + \mathscr{A}\alpha_2 \in W_1 + W_2$ |
| **P3** | 交的分配不成立  | $W \cap (W_1 + W_2) = (W \cap W_1) + (W \cap W_2)$ **一般不成立**                                                  | 反例：$V = \mathbb{R}^2$，三条不同直线                                                                                 |
| **P4** | 直和的不变继承  | $W_1, W_2$ 是 $\mathscr{A}$-子空间且 $W_1 \cap W_2 = \{0\}$ $\Rightarrow$ $W_1 \oplus W_2$ 是 $\mathscr{A}$-子空间     | 由 P2，$W_1 + W_2 = W_1 \oplus W_2$                                                                            |
| **P5** | 特征子空间的直和 | $\lambda_1 \neq \lambda_2$ $\Rightarrow$ $V_{\lambda_1} + V_{\lambda_2} = V_{\lambda_1} \oplus V_{\lambda_2}$ | $V_{\lambda_1} \cap V_{\lambda_2} = \{0\}$（特征子空间性质）                                                          |
| **P6** | 商空间诱导变换  | $W$ 是 $\mathscr{A}$-子空间 $\Rightarrow$ $\mathscr{A}$ 在 $V/W$ 上诱导线性变换 $\bar{\mathscr{A}}$                       | $\bar{\mathscr{A}}(\alpha + W) = \mathscr{A}\alpha + W$（良定性由 $W$ 不变保证）                                       |
| **P7** | 不变子空间与矩阵 | $V = W_1 \oplus \cdots \oplus W_k$（$W_i$ 皆 $\mathscr{A}$-不变）$\Leftrightarrow$ $\mathscr{A}$ 的矩阵为分块对角形         | 取各 $W_i$ 的基合并为 $V$ 的基                                                                                        |
| **P8** |  多项式保持性  | $W$ 是 $\mathscr{A}$-子空间 $\Rightarrow$ $W$ 是 $f(\mathscr{A})$-子空间（$\forall f \in P[x]$）                        | $f(\mathscr{A}) = a_m\mathscr{A}^m + \cdots + a_0\mathscr{E}$，每项保持 $W$                                       |

## 四、不变子空间与矩阵简化的核心定理

### 定理1：单不变子空间 → 分块上三角

设 $W \subseteq V$ 是 $\mathscr{A}$ 的非平凡不变子空间，$\dim W = k$。取 $W$ 的基 $\{\varepsilon_1, \ldots, \varepsilon_k\}$，扩充为 $V$ 的基 $\{\varepsilon_1, \ldots, \varepsilon_k, \varepsilon_{k+1}, \ldots, \varepsilon_n\}$。则 $\mathscr{A}$ 在此基下的矩阵为

$$A = \begin{pmatrix} A_1 & B \\ 0 & A_2 \end{pmatrix}$$

其中 $A_1$ 是 $k \times k$ 矩阵（即 $\mathscr{A}|_W$ 的矩阵），$A_2$ 是 $(n-k) \times (n-k)$ 矩阵。

> **原理**：$W$ 中向量被 $\mathscr{A}$ 作用后仍在 $W$ 中，故前 $k$ 个基向量的像只用前 $k$ 个基向量表示，下方为零。

---

### 定理2：多不变子空间直和 → 分块对角

设 $V = W_1 \oplus W_2 \oplus \cdots \oplus W_k$，每个 $W_i$ 都是 $\mathscr{A}$-不变子空间，$\dim W_i = n_i$。取各 $W_i$ 的基合并为 $V$ 的基，则 $\mathscr{A}$ 的矩阵为

$$A = \begin{pmatrix} A_1 & & \\ & \ddots & \\ & & A_k \end{pmatrix}$$

其中 $A_i$ 是 $n_i \times n_i$ 矩阵（即 $\mathscr{A}|_{W_i}$ 的矩阵）。

> **这是将复杂变换"分而治之"的数学实现**：每个 $A_i$ 独立描述 $\mathscr{A}$ 在 $W_i$ 上的行为。

---

### 定理3：特征子空间直和与对角化

$$\mathscr{A} \text{ 可对角化} \Leftrightarrow V = V_{\lambda_1} \oplus V_{\lambda_2} \oplus \cdots \oplus V_{\lambda_s}$$

即：全空间可分解为特征子空间的直和。

| 对角化情形 | Jordan 标准型情形 |
|:---:|:---:|
| $V = V_{\lambda_1} \oplus \cdots \oplus V_{\lambda_s}$ | $V = G_{\lambda_1} \oplus \cdots \oplus G_{\lambda_s}$ |
| 每个 $V_{\lambda_i}$ 是特征子空间 | 每个 $G_{\lambda_i}$ 是根子空间 |
| $\dim V_{\lambda_i} = m_i$（代数重数） | $\dim G_{\lambda_i} = m_i$ |
| $\mathscr{A}|_{V_{\lambda_i}} = \lambda_i \mathscr{E}$ | $\mathscr{A}|_{G_{\lambda_i}}$ 有 Jordan 型 |
| 矩阵为纯对角形 | 矩阵为分块对角形（含 Jordan 块） |

---

## 五、四大核心应用

### 应用1：将矩阵化为分块对角形（降维打击）

**问题**：给定复杂的 $n \times n$ 矩阵 $A$，研究其行为。

**策略**：找非平凡不变子空间 $W$，将 $V$ 分解。若 $V = W_1 \oplus W_2$，则

$$A = \begin{pmatrix} A_1 & 0 \\ 0 & A_2 \end{pmatrix}$$

$A_1, A_2$ 阶数更低，可分别研究。

**例子**：$A = \begin{pmatrix} 2 & 0 & 0 \\ 0 & 3 & 1 \\ 0 & 0 & 3 \end{pmatrix}$

- $W_1 = \text{span}\{e_1\}$：$Ae_1 = 2e_1 \in W_1$，是不变子空间
- $W_2 = \text{span}\{e_2, e_3\}$：$Ae_2 = 3e_2 \in W_2$，$Ae_3 = e_2 + 3e_3 \in W_2$，是不变子空间

$V = W_1 \oplus W_2$，$A_1 = (2)$，$A_2 = \begin{pmatrix} 3 & 1 \\ 0 & 3 \end{pmatrix} = J(3, 2)$

---

### 应用2：对角化的判定与构造

$$\mathscr{A} \text{ 可对角化} \Leftrightarrow \sum_{i=1}^s \dim V_{\lambda_i} = n$$

**操作**：
1. 求所有特征值 $\lambda_1, \ldots, \lambda_s$
2. 求各特征子空间 $V_{\lambda_i}$ 的维数（几何重数）
3. 若维数和为 $n$，则 $V = \bigoplus V_{\lambda_i}$
4. 取各 $V_{\lambda_i}$ 的基合并，即得对角化基

---

### 应用3：Jordan 标准型的构造

**根子空间分解**：在代数闭域 $\mathbb{C}$ 上

$$V = G_{\lambda_1} \oplus G_{\lambda_2} \oplus \cdots \oplus G_{\lambda_s}$$

其中 $G_\lambda = \{\alpha \mid (\mathscr{A} - \lambda\mathscr{E})^k\alpha = 0 \text{（某 }k\text{）}\}$ 是根子空间。

**在 $G_\lambda$ 内部**：通过循环不变子空间的直和分解，得到 Jordan 链。

| 层次 | 分解 | 不变子空间类型 |
|:---:|:---:|:---:|
| 第一层 | $V = \bigoplus_i G_{\lambda_i}$ | 根子空间（按特征值） |
| 第二层 | $G_\lambda = \bigoplus_j C_j$ | 循环不变子空间（Jordan 链） |
| 第三层 | $C_j = \text{span}\{\xi, (\mathscr{A}-\lambda\mathscr{E})\xi, \ldots\}$ | Jordan 链 |

---

### 应用4：最小多项式的结构分析

$$V = \ker(\mathscr{A} - \lambda_1\mathscr{E})^{m_1} \oplus \cdots \oplus \ker(\mathscr{A} - \lambda_s\mathscr{E})^{m_s}$$

其中 $m(\lambda) = (\lambda - \lambda_1)^{m_1} \cdots (\lambda - \lambda_s)^{m_s}$ 是最小多项式。

每个 $\ker(\mathscr{A} - \lambda_i\mathscr{E})^{m_i}$ 就是根子空间 $G_{\lambda_i}$，是 $\mathscr{A}$-不变子空间。

---

## 六、完整计算示例

### 例题

设 $\mathscr{A} : \mathbb{R}^3 \to \mathbb{R}^3$，在标准基下的矩阵为

$$A = \begin{pmatrix} 3 & 1 & 0 \\ 0 & 3 & 0 \\ 0 & 0 & 2 \end{pmatrix}$$

(1) 求 $A$ 的所有非平凡不变子空间；

(2) 判断是否存在补不变子空间，即 $V = W_1 \oplus W_2$（$W_1, W_2$ 皆不变）；

(3) 若存在，将 $A$ 化为分块对角形。

---

### 解答

#### (1) 求非平凡不变子空间

**Step 1：求特征值**

$A$ 是上三角矩阵，特征值 = 对角元：$\lambda_1 = 3$（代数重数 2），$\lambda_2 = 2$（代数重数 1）。

**Step 2：求 $V_3$**

$(3E - A)X = 0$：

$$3E - A = \begin{pmatrix} 0 & -1 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 1 \end{pmatrix}$$

方程组：$-x_2 = 0$，$x_3 = 0$ $\Rightarrow$ $x_2 = 0, x_3 = 0$，$x_1$ 自由。

$$V_3 = \text{span}\{(1, 0, 0)'\}, \quad \dim V_3 = 1$$

几何重数 $g_3 = 1 < m_3 = 2$，**不可对角化**。

**Step 3：求 $V_2$**

$(2E - A)X = 0$：

$$2E - A = \begin{pmatrix} -1 & -1 & 0 \\ 0 & -1 & 0 \\ 0 & 0 & 0 \end{pmatrix}$$

方程组：$-x_1 - x_2 = 0$，$-x_2 = 0$ $\Rightarrow$ $x_2 = 0, x_1 = 0$，$x_3$ 自由。

$$V_2 = \text{span}\{(0, 0, 1)'\}, \quad \dim V_2 = 1$$

**Step 4：找其他不变子空间**

由 $A$ 的分块结构，$W = \text{span}\{e_1, e_2\} = \{(x, y, 0)'\}$：

- $Ae_1 = 3e_1 \in W$
- $Ae_2 = e_1 + 3e_2 \in W$

$W$ 是 $A$-不变子空间（前 two 维子空间）。

**全部非平凡不变子空间**：

| 不变子空间 | 维数 | 说明 |
|:---:|:---:|:---|
| $V_3 = \text{span}\{e_1\}$ | 1 | 特征子空间（$\lambda = 3$） |
| $V_2 = \text{span}\{e_3\}$ | 1 | 特征子空间（$\lambda = 2$） |
| $W = \text{span}\{e_1, e_2\}$ | 2 | 前二维子空间 |

> 注意：并非任意子空间都是不变子空间！例如 $W' = \text{span}\{e_1, e_3\}$ 不是（$Ae_2 \notin W'$），$U = \text{span}\{e_2, e_3\}$ 也不是（$Ae_1 \notin U$）。需要逐一验证。

---

#### (2) 补不变子空间

**检验 $V = W \oplus V_2$**：

- $W = \text{span}\{e_1, e_2\}$，$V_2 = \text{span}\{e_3\}$
- $W \cap V_2 = \{0\}$（$e_3$ 的第三分量为1，$W$ 中向量第三分量为0）
- $\dim W + \dim V_2 = 2 + 1 = 3 = \dim V$

**$V = W \oplus V_2$**，且 $W, V_2$ 都是 $A$-不变子空间。$\checkmark$

---

#### (3) 分块对角形

在基 $\{e_1, e_2, e_3\}$（标准基）下，$A$ 已经是：

$$A = \begin{pmatrix} \underbrace{\begin{matrix} 3 & 1 \\ 0 & 3 \end{matrix}}_{A_1} & \begin{matrix} 0 \\ 0 \end{matrix} \\ \begin{matrix} 0 & 0 \end{matrix} & \underbrace{(2)}_{A_2} \end{pmatrix}$$

即 $A = \begin{pmatrix} J(3,2) & 0 \\ 0 & (2) \end{pmatrix}$。

在 $W$ 上：$\mathscr{A}|_W$ 的矩阵是 $J(3, 2)$（不可对角化的 Jordan 块）

在 $V_2$ 上：$\mathscr{A}|_{V_2}$ 的矩阵是 $(2)$（数乘变换）

> **关键观察**：虽然 $A$ 整体不可对角化，但通过不变子空间分解，我们将其分离为一个"坏"的部分（$J(3,2)$）和一个"好"的部分（$(2)$）。这就是不变子空间"分而治之"的力量。

## 七、不变子空间与全书知识的关联网络

```
不变子空间 W（A W \subseteq W）
    |
    |-- 第4章：矩阵分块
    |       \-- W 的基扩充为 V 的基 --> 矩阵分块上三角化
    |
    |-- 第5章：二次型
    |       \-- 正定矩阵的分解 A = C'C
    |       \-- 惯性定理中的正/负定子空间
    |
    |-- 第6章：线性空间
    |       \-- 子空间、直和分解 V = W_1 \oplus W_2
    |       \-- 商空间 V/W 与诱导变换

    |-- 第7章（核心）：
    |       |-- 特征子空间 V_\lambda（最重要的一类不变子空间）
    |       |-- 核 A^{-1}(0)、像 AV
    |       |-- 根子空间 G_\lambda（Jordan 标准型的基础）

            \-- 谱定理：V = \oplus V_{\lambda_i}（正交直和）
```

## 八、不变子空间 vs 特征子空间：关键对比

| 对比维度 | 不变子空间 | 特征子空间 |
|:---:|:---:|:---:|
| **定义** | $\mathscr{A}W \subseteq W$（一般） | $W = V_\lambda = \{\alpha \mid \mathscr{A}\alpha = \lambda\alpha\}$（特殊） |
| **关系** | 一般概念 | 不变子空间的**特例** |
| **变换行为** | $\mathscr{A}$ 在 $W$ 上任意线性变换 | $\mathscr{A}$ 在 $V_\lambda$ 上是**数乘** $\lambda\mathscr{E}$ |
| **矩阵形状** | 分块上三角（单）/ 分块对角（多） | 对角元为 $\lambda$ 的纯对角块 |
| **构造难度** | 一般较难找 | 解 $(\lambda E - A)X = 0$ 系统方法 |
| **直和分解** | $V = W_1 \oplus W_2$（需要构造） | $V = \bigoplus V_{\lambda_i}$（特征值自动给出） |
| **与对角化关系** | 存在分解为1维不变子空间的直和 <-> 可对角化 | 存在分解为特征子空间的直和 <-> 可对角化 |

**核心关系**：

$$\text{特征子空间} \subsetneq \text{不变子空间}$$

每个特征子空间都是不变子空间，但不变子空间不一定是特征子空间。

## 九、常见误区辨析

| 误区 | 正确理解 |
|:---|:---|
| **"不变子空间 = 特征子空间"** | 特征子空间是不变子空间的**特例**。不变子空间只要求 $\mathscr{A}\alpha \in W$，不要求 $\mathscr{A}\alpha = \lambda\alpha$ |
| **"任意子空间都是不变子空间"** | 必须验证 $\mathscr{A}W \subseteq W$。反例：$A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$，$W = \text{span}\{e_2\}$ 不是不变子空间（$Ae_2 = e_1 \notin W$） |
| **"不变子空间的补空间也是不变的"** | **一般不成立**！反例：$A = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}$，$W = \text{span}\{e_1\}$ 是不变子空间，但其补 $\text{span}\{e_2\}$ 不是 |
| **"V = AV \oplus A^{-1}(0)"** | 一般不成立。维数定理只说 $\dim(AV) + \dim(A^{-1}(0)) = n$，不保证直和。反例：$A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$，$AV = A^{-1}(0) = \text{span}\{e_1\}$ |
| **"不变子空间的维数任意"** | 受限制。如特征子空间的维数（几何重数）$\leq$ 代数重数 |

## 十、一句话总结

> **不变子空间**是线性变换 $\mathscr{A}$ 作用下"自我封闭"的子空间。它是将复杂变换"分而治之"的核心工具——通过直和分解 $V = W_1 \oplus \cdots \oplus W_k$，$\mathscr{A}$ 的矩阵化为分块对角形，每个块对应一个更简单的子变换。从特征子空间到根子空间，不变子空间贯穿着对角化理论和 Jordan 标准型的整个构造过程。
